# Git & GitHub

A beginner-friendly repository for learning the basics of Git and GitHub.

## 📖 About

This repository contains notes, examples, and practice commands for Git and GitHub. It is intended for beginners who want to learn version control and collaborate using GitHub.

## 📚 Topics Covered

- What is Git?
- What is GitHub?
- Git Installation
- Git Configuration
- Initializing a Repository
- Staging Changes
- Creating Commits
- Branching
- Merging
- Cloning Repositories
- Push & Pull
- Git Remote
- Git Ignore
- GitHub Workflow

## 🛠️ Basic Git Commands

```bash
git init
git add .
git commit -m "Initial commit"
git status
git log
git branch
git checkout
git merge
git remote -v
git push
git pull
```

## 🎯 Purpose

The main goal of this repository is to practice Git commands and understand how Git and GitHub work together in real-world development.
